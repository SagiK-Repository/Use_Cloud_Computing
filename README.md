## 클라우딩 컴퓨터?
- 클라우딩 컴퓨터란, 온라인에서 컴퓨터 처럼 활용할 수 있는 서비스이다.
- 사양 좋은 GPU등 여러가지 활용이점이 있다.
- 작게는 리눅스, 크게는 여러 활용도가 존재한다.
